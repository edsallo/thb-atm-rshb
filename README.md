# THB ATM RSHB

Калькулятор снятия наличных тайских батов с карты UnionPay Россельхозбанка.

## Возможности

- Получение актуального курса UnionPay (CNY → THB)
- Автоматическое использование вчерашнего курса, если сегодняшний еще недоступен
- Учет комиссии тайского банкомата
- Учет комиссии Россельхозбанка
- Запуск из Scriptable
- Поддержка запуска из Apple Shortcuts

## План развития

- История курса
- Виджет Scriptable
- Поддержка рублевой карты РСХБ
- Интеграция с Shortcuts
