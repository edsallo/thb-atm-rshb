# THB ATM RSHB

Калькулятор Scriptable для оценки снятия тайских батов (THB) в банкомате Таиланда с UnionPay-карты Россельхозбанка.

Скрипт поддерживает два сценария:

- **рублёвая карта (RUB):** THB конвертируются UnionPay в CNY, затем РСХБ списывает RUB по курсу для операций **вне сети Банка**;
- **юаневая карта (CNY):** со счёта списываются CNY; дополнительно показывается ориентир, сколько RUB нужно для покупки этой суммы CNY по курсу операций **в сети Банка**.

## Что учитывается

- выдаваемая банком сумма THB;
- фиксированная доплата тайского ATM — **250 THB**;
- дневной курс UnionPay `THB → CNY`;
- комиссия РСХБ за снятие в стороннем банкомате:
  - RUB-счёт — **1,5%, минимум 250 RUB**;
  - CNY-счёт — **4%, минимум 70 CNY**;
- отдельные карточные курсы РСХБ:
  - `temporaryRates` — операции **вне сети Банка**: используется для RUB-карты при снятии за рубежом;
  - `ratescards` — операции **в сети Банка**: используется для оценки покупки CNY.

Курсы кэшируются локально: UnionPay — на 6 часов, РСХБ — на 15 минут. При сбое сети допускается использование кэша сроком до 7 дней.

## Установка в Scriptable

1. Откройте [THB-ATM-RSHB.js](./THB-ATM-RSHB.js) на iPhone или iPad.
2. Скопируйте содержимое в новый скрипт приложения [Scriptable](https://scriptable.app/).
3. Запустите скрипт, выберите RUB- или CNY-карту и укажите сумму выдачи.

По умолчанию предложено `30 000 THB`.

## Apple Shortcuts

Передайте скрипту текстовый параметр:

```text
rub 30000
cny 30000
rub 30000 refresh
cny 30000 json
```

- `rub` / `cny` — тип счёта карты;
- число — сумма, которую должен выдать банкомат, в THB;
- `refresh` — игнорировать свежий кэш;
- `json` — вернуть структурированный результат для Shortcuts.

## Источники

- [Дневные курсы UnionPay](https://www.unionpayintl.com/upload/jfimg/)
- [Карточные курсы РСХБ](https://www.rshb.ru/natural/transfers/kurs-pokupki-prodazhi-dlya-operaciy-s-ispolzovaniem-platezhnyh-kart)

Расчёт является ориентировочным: окончательная сумма зависит от курса на дату проведения операции, условий РСХБ и фактической доплаты конкретного банкомата Таиланда.
